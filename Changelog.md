# PRISM Changelog (Public Summary)

**Project:** PRISM (Prospect Research and Intellegence System for Major gifts)  
**Author:** Jacob Sykora

**Purpose:**  
PRISM is a proprietary data management and query automation toolkit jointly developed by Jacob Sykora and the Archdiocese of Omaha.  
This public changelog summarizes the major functional milestones and structural improvements in each version without disclosing proprietary implementation details.

---

## Version 0.1  
- Initial prototype released  
- Implemented core data query and validation framework  
- Added resilience to prevent rate-limit and session errors  
- Introduced worksheet-agnostic operation and session resume capability  

## Version 0.2  
- Improved data normalization and matching across datasets  
- Refactored internal query architecture for modularity and reusability  

## Version 0.3  
- Added support for automated web navigation within dynamic, JavaScript-based pages  
- Introduced an experimental library for structured data lookups from public web portals  

## Version 0.4  
- Expanded support for JSON-based data parsing with better handling of incomplete or inconsistent datasets  

## Version 0.5  
- Enhanced data aggregation and export workflow  
- Introduced a simple user start menu for streamlined access  

## Version 0.6  
- Rebuilt internal logic for improved performance and adaptability across data sources  
- Converted toolkit into an Excel Add-in with initial auto-installation support  

## Version 0.7  
- Introduced a prototype for an additional regional data connector  
- Improved data preview extraction and standardized output formatting  

## Version 0.8  
- Refined table combination and filtering logic  
- Expanded compatibility with previously unsupported dataset structures  

## Version 0.9  
- Overhauled HTML-based data extraction to uncover and parse deeply nested information  
- Standardized all outputs into the PRISM parcel data format
